Typing Speed Game built using Streamlit. Players select a difficulty level and type the displayed sentence while the app tracks elapsed time, WPM, and accuracy. High scores are stored locally and ranked by performance.
Features:
- Difficulty modes: Easy / Medium / Hard
- Live WPM and accuracy computation
- Highlights correct and incorrect words
- Persistent high score leaderboard (text file storage)
- Simple UI for quick testing and fun challenges
